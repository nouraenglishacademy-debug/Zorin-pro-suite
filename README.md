# Zorin OS Pro Suite Installer

أداة برمجية متقدمة ومفتوحة المصدر مصممة خصيصاً لمستخدمي Zorin OS Lite (بيئة XFCE)، تهدف إلى أتمتة عملية تخصيص النظام وتثبيت حزمة برامج احترافية بضغطة زر واحدة.

## المميزات الرئيسية
* **تخصيص بصري:** سحب وتثبيت أحدث ثيمات الأيقونات (مثل Orchis و Tela) مباشرة من المصادر المفتوحة[span_1](start_span)[span_1](end_span).
* **أتمتة ذكية:** تثبيت وتجهيز 25 برنامجاً أساسياً للمطورين والمصممين والمستخدمين العامين[span_2](start_span)[span_2](end_span).
* **واجهة رسومية (GUI):** معالج تثبيت رسومي مبني باستخدام أداة `yad` لتوفير تجربة مستخدم سهلة.
* **دعم النظام:** إعداد تلقائي لتوسيط شريط المهام (Windows 11 Layout) وتفعيل شريط (Plank Dock)[span_3](start_span)[span_3](end_span).

## التثبيت
يمكنك تثبيت الأداة كحزمة نظام رسمية عبر الخطوات التالية:

1. قم بتحميل أحدث إصدار من ملف **zorin-pro-suite_1.0.1_all.deb** من قسم [Releases].
2. قم بفتح التيرمنال في مكان تحميل الملف ونفذ الأمر التالي:
```bash
   sudo apt install ./zorin-pro-suite_1.0.1_all.deb
