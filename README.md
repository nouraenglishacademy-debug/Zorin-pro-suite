# 🚀 Zorin OS Pro Suite
Zorin OS Pro Suite هو معالج تثبيت وتخصيص آلي لنظام Zorin OS Lite. تتيح الأداة تخصيص واجهة النظام عبر تثبيت ثيمات وأيقونات عصرية، مع أتمتة تثبيت 25 برنامجاً أساسياً للمطورين بضغطة زر. توفر الأداة واجهة رسومية سهلة الاستخدام، وتدعم تهيئة شريط المهام لمحاكاة Windows 11 أو macOS.
### 🛠️ المميزات:
 * **تخصيص جمالي:** سحب وتثبيت ثيمات وأيقونات عصرية من الويب.
 * **أتمتة البرامج:** تثبيت 25 تطبيقاً أساسياً بضغطة زر واحدة.
 * **تحكم ذكي:** واجهة رسومية مبنية بـ yad لتسهيل الاختيارات.
### 📦 التثبيت:
للبدء، قم بتحميل ملف zorin-pro-suite_1.0.1_all.deb ثم نفذ الأمر التالي في التيرمنال:
```bash
sudo apt install ./zorin-pro-suite_1.0.1_all.deb

```
